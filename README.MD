# Design

![Design](design.png)


Houba gravitace – po jejím snědení se na pár sekund obrátí gravitace a postavy chodí po stropě.
Skákací houba – dává medvědovi nebo kočce super skok (dvojnásobná výška nebo dvojitý skok).
Houba miniaturizace – medvěd se na chvíli zmenší a vleze do malých prostor.
Houba lenosti – zpomalí pohyb postavy, jako by byla unavená.
Houba iluze – na pár sekund se objeví falešné plošiny nebo nepřátelé.
Houba magnetická – přitahuje kovové předměty (klíče, plošiny apod.).
Houba neviditelnosti – postava zmizí a hráč musí hádat, kde je.
Houba klonovací – vytvoří stínový obraz postavy, který opakuje pohyby.
Houba zpětného času – vrátí postavu pár sekund zpět v čase.
